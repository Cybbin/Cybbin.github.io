JavaScript数组常用方法汇总

concat()

用法：连接两个或更多的数组，并返回结果

参数 描述
array 必需。该参数可以是具体的值，也可以是数组对象，可以是任意多个

copyWithin()

用法：从数组的指定位置拷贝元素到数组的另一个指定位置中

参数 描述
target 必需。复制到指定目标索引位置
start 必需。元素复制的起始位置
end 可选。停止复制的索引位置，默认为 array.length

every()

用法：检测数组所有元素是否都符合指定条件（通过函数提供）

参数 描述
function 必需。函数，数组中每个元素都会执行这个函数

注：

  1. 如果数组中检测到一个元素不满足，则整个表达式返回false，且剩下的元素不会再进行检测
  2. 如果所有元素都满足条件，则返回true

–

fill()

用法：将一个固定值替换数组的所有元素。
|参数 |描述|
|:—-:|:–:|
|value|必需。填充的值|
|start|可选。开始填充的位置|
|end|可选。停止填充的位置，默认为 array.length|

filter()

用法：创建一个新的数组，新数组中的元素是通过检查指定数组中符合条件的所有元素

参数 描述
function 必需。函数，数组中每个元素都会执行这个函数

find()

用法：获取数组第一个符合条件的元素（传入函数）

注：

  1. 当数组中的元素在测试条件时返回true时，find()返回符合条件的元素，之后的值不会再调用执行函数
  2. 如果没有符合条件的元素返回 undefined
参数 描述
function 必需。函数，数组中每个元素都会执行这个函数

findIndex()

用法：返回符合条件的第一个元素位置

参数 描述
function 必需。函数，数组中每个元素都会执行这个函数

注：

  1. 当数组中的元素在测试条件时返回 true 时, findIndex() 返回符合条件的元素的索引位置，之后的值不会再调用执行函数
  2. 如果没有符合条件的元素返回-1

forEach()

用法：遍历数组

参数 描述
function 必需。函数，数组中每个元素都会执行这个函数。
函数参数 描述
currentValue 必需。当前元素。
index 可选。当前元素的索引值。
arr 可选。当前元素所属的数组对象。

indexOf()

用法：返回某个指定的字符串值在字符串中首次出现的位置

参数 描述
item 必需。查找的元素
start 可选。开始检索的位置，如果省略，则从字符串的首字符开始检索

注：

  1. 该方法将从头到尾地检索字符串 stringObject，看它是否含有子串 searchvalue。开始检索的位置在字符串的 fromindex 处或字符串的开头（没有指定 fromindex 时）。如果找到一个 searchvalue，则返回 searchvalue 的第一次出现的位置。
  2. 如果在数组中没找到字符串则返回 -1。

join()

用法：把数组的所有元素放入一个字符串。元素通过指定的分隔符进行分隔。

参数 描述
separator 可选。指定要使用的分隔符，如果省略该参数，则使用逗号作为分隔符

lastIndexOf()

用法：返回某个指定的字符串值在字符串中最后出现的位置

参数 描述
item 必需。查找的元素
start 可选。从后往前开始检索的位置，如果省略，则从字符串的最后一个字符处开始检索

注：

  1. 可返回一个指定的字符串值最后出现的位置，在一个字符串中的指定位置从后向前搜索。
  2. 如果在数组中没找到字符串则返回 -1。

map()

用法：返回一个新数组，数组中的元素为原始数组调用函数处理后的值

参数 描述
function 必需。函数，数组中每个元素都会执行这个函数。
函数参数 描述
currentValue 必需。当前元素。
index 可选。当前元素的索引值。
arr 可选。当前元素所属的数组对象。

pop()

用法：删除并返回数组的最后一个元素

push()

用法：向数组的末尾添加一个或更多元素，并返回新的长度

参数 描述
newElement 必需。要添加到数组的元素，可以是任意多个

reduce()

用法：接收一个函数作为累加器，数组中的每个值（从左到右）开始缩减，最终计算为一个值

参数 描述
function 必需。函数，数组中每个元素都会执行这个函数。
函数参数 描述
total 必需。初始值，或计算结束后的返回值。
currentValue 必需。当前元素。
index 可选。当前元素的索引值。
arr 可选。当前元素所属的数组对象。

reduce()

用法：接收一个函数作为累加器，数组中的每个值（从右到左）开始缩减，最终计算为一个值

参数 描述
function 必需。函数，数组中每个元素都会执行这个函数。
函数参数 描述
total 必需。初始值，或计算结束后的返回值。
currentValue 必需。当前元素。
index 可选。当前元素的索引值。
arr 可选。当前元素所属的数组对象。

reverse()

用法：颠倒数组中元素的顺序

shift()

用法：删除并返回数组的第一个元素

unshift()

用法：向数组的开头添加一个或更多元素，并返回新的长度

参数 描述
newElement 必需。要添加到数组的元素，可以是任意多个

slice()

用法：从已有的数组返回选定区间的元素

参数 描述
start 必需。规定从何处开始选取，如果是负数，则从数组尾部开始算起，即-1指最后一个元素，-2指倒数第二个元素，以此类推
end 可选。规定从何处结束选取，如果没有指定该参数，那么切分的数组包含从start到end-1的所有元素，如果是负数，则从数组尾部开始算起

some()

用法：检测数组中的元素是否满足指定条件（函数提供）
|参数 |描述|
|:—-:|:–:|
|function|必需。函数，数组中每个元素都会执行这个函数|

函数参数 描述
currentValue 必需。当前元素。
index 可选。当前元素的索引值。
arr 可选。当前元素所属的数组对象。

sort()

用法：对数组的元素进行排序

参数 描述
sortby 可选。规定排序顺序。必须是函数，如果省略该参数，则按照字符编码的顺序进行排序

splice()

用法：删除元素，并向数组添加新元素

参数 描述
index 必需。规定添加/删除的位置，如果是负数，则从数组尾部开始算起
howmany 必需。要删除的数量，如果设置为0，则不会删除
newElement 可选。向数组添加的新元素

toString()

用法：把数组转换为字符串，并返回结果，返回值与没有参数的join()方法返回的字符串相同

valueOf()

用法：返回Array对象的原始值
