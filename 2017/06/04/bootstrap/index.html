bootstrap源码分析

发表于 2017-06-04 | 分类于 源码 | | 阅读次数

记录分析bootstrap源码

导入

栅格系统

.container

margin:0 auto + padding:0 15px + clearfix + media

  1. margin:0 auto 让 container 居中;
  2. padding:0 15px 让 contianer 两边有一定距离;
  3. clearfix 为了防止与BFC有关的一些BUG;
  4. media 响应式，根据不同的屏幕大小设置宽度。
    源码：

注：container-fluid 跟 container 唯一区别是没有响应式。

.row

clearfix + margin:0 -15px

  1. clearix 是为了包含浮动的 col
  2. 负的 margin 消除 container 的padding

.col

float:left + position:relative + padding:0 15px + width

  1. float:left 一行放多个div元素的原理，让 col-xs-offset 生效，原理为 margin-left;
  2. position：relative 让 col-xs-pull 和 col-xs-push 生效，原理为 left,right;
  3. padding 为了让每个元素有间隔，不用担心元素占据更多控件，原理为 box-sizing:border-box;
  4. media 让元素能根据不同的屏幕大小显示不同样式。

.col-xs-offset

margin-left
向右偏移，不能交换位置

.col-xs-push .col-xs-pull

col-xs-push : left , col-xs-pull : right
通过设置right,left实现定位效果，能交换位置

表单

.form-inline

display:inline-block + vertical-align:middle + width:auto

.form-inline 作用是控制子元素中的 .form-control 的 display 为 inline-block

.form-horizontal

form-horizontal > .form-group = clearfix
form-horizonotal > .control-label = text-align:right

.form-horizontal 作用是让表单元素可以使用栅格系统，无需加 row

disabled

cursor:not-allowed

disabled 作用是禁用表单元素

图片

.img-responsive

max-width:100% + height:auto + display:block

.img-responsive 可以让图片支持响应式布局。

注: 如果需要让使用了 .img-responsive 类的图片水平居中，请使用 .center-block 类，不要用 .text-center。
